pub struct FileOpenConfig {
    pub path: Path,
    pub flags: int,
    pub mode: int,
}

Data needed to make a successful open(2) call Using unix flag conventions for now, which happens to also be what's supported libuv (it does translation to windows under the hood).

Fields

path

Path to file to be opened

flags

Flags for file access mode (as per open(2))

mode

File creation mode, ignored unless O_CREAT is passed as part of flags
