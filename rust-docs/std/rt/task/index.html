Module std::rt::task[src]

Implementations of language-critical runtime features like @. Language-level runtime services that should reasonably expected to be available 'everywhere'. Local heaps, GC, unwinding, local storage, and logging. Even a 'freestanding' Rust would likely want to implement this.

Task

The Task struct represents all state associated with a rust task. There are at this point two primary "subtypes" of task, however instead of using a subtype we just have a "task_type" field in the struct. This contains a pointer to another struct that holds the type-specific state.
