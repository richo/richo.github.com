pub trait FromVec<T> {
    fn from_vec(v: Vec<T>) -> Self;
}

Mechanism to convert from a Vec<T> to a [T].

In a post-DST world this will be used to convert to any Ptr<[T]>.

This could be implemented on more types than just pointers to vectors, but the recommended approach for those types is to implement FromIterator.
